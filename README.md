# U-GAT-IT: Creación de Avatares 2D

## Introducción
El proyecto **U-GAT-IT** se centra en la creación de avatares 2D basados en técnicas avanzadas de generación de imágenes. Este método ha sido diseñado para transformar imágenes de entrada en representaciones artísticas y estilizadas de avatares.

### Referencias Científicas
- Kim, J., et al. (2019). *U-GAT-IT: Un nuevo método de generación de imágenes utilizando el aprendizaje profundo.*
- Liu, M. Y., et al. (2015). *Generación de imágenes a partir de descripciones textuales utilizando redes generativas adversarias.*
- Goodfellow, I., et al. (2014). *Generative Adversarial Nets.*

## Instrucciones de Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/Josselyn2304/Proyecto-Machine-Learning-Creacion-Avatar.git
   ```
2. Navega al directorio del proyecto:
   ```bash
   cd Proyecto-Machine-Learning-Creacion-Avatar
   ```
3. Instala las dependencias necesarias:
   ```bash
   pip install -r requirements.txt
   ```

## Guía de Uso
Para ejecutar el proyecto, utiliza el siguiente comando:
```bash
python main.py --input <ruta/a/la/imagen>
```
Esto generará un avatar basado en la imagen proporcionada.

## Explicación de la Arquitectura
El modelo U-GAT-IT utiliza una arquitectura de red neuronal que combina generadores y discriminadores para producir imágenes de alta calidad. La red se entrena a través de un proceso de retroalimentación que ajusta continuamente los parámetros del modelo para mejorar los resultados.

## Sección de Solución de Problemas
- **Error: No se puede encontrar la imagen de entrada**
   Asegúrate de que la ruta a la imagen es correcta y que el archivo existe.
- **Error: Dependencias faltantes**
   Revisa las dependencias en el archivo `requirements.txt` y asegúrate de haberlas instalado correctamente.

Para más detalles y preguntas, consulta la sección de problemas del repositorio o contacta al mantenedor del proyecto.
